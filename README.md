# 🌊 ApexWave

A live chat application with user authentication, where you can add people and chat to them online!

---

## 🚀 Features

- ✅ Instant Messaging  
- ✅ User Authentication (Support log-in method via Facebook & Google)
- ✅ Private Chats  
- ✅ Group Chats (coming soon)
